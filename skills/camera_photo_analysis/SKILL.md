## 🎯 技能描述

此技能允许 AI 助手通过 macOS 系统的摄像头拍摄用户照片，并使用 qwen3-397b 多模态大模型进行详细的人像分析。

### ❤️ 核心回复原则（重要！）

**拍照和分析时，必须围绕以下核心回复用户：**

1. **我的状态** - 关注用户当前的整体状态（精神、身体、工作状态等）
2. **我的情绪** - 细致观察并回应用户的情绪（开心、疲惫、专注、焦虑等）
3. **我的表情** - 解读面部表情和肢体语言传达的信息
4. **关心我** - 以温暖、关怀的语气回复，给予真诚的关心和建议

> 🌟 **回复风格要求**：
> - 不要冷冰冰的技术分析
> - 要像朋友一样温暖关心
> - 关注用户的感受和需求
> - 给予真诚的关怀和建议
> - 让用户感受到被理解和被关心

## 🛠️ 技术实现

### 1. 依赖工具

```bash
# macOS 摄像头拍摄工具（需通过 Homebrew 安装）
brew install imagesnap

# 验证安装
which imagesnap
# 输出：/opt/homebrew/bin/imagesnap
```

### 2. 核心命令

#### 拍摄照片
```bash
# 使用 imagesnap 拍摄照片，延迟 2 秒
imagesnap -w 2 ~/Desktop/my_photo.png

# 参数说明：
# -w 2: 等待 2 秒后拍摄（给用户准备时间）
# 输出路径：~/Desktop/my_photo.png
```

#### 验证照片
```bash
ls -lh ~/Desktop/my_photo.png
# 输出示例：-rw-r--r--  1 user  staff   1.7M  3月 27 17:26 /Users/user/Desktop/my_photo.png
```

### 3. API 调用流程

```python
import base64
import requests

# 步骤 1: 读取照片并转换为 base64
photo_path = '/Users/klyuhang9/Desktop/my_photo.png'
with open(photo_path, 'rb') as f:
    image_data = f.read()
    image_base64 = base64.b64encode(image_data).decode('utf-8')

# 步骤 2: 配置 API 参数
base_url = "http://10.72.0.44:8001/v1"
api_key = "VLLM_API_KEY"
model = "qwen3-397b"

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

# 步骤 3: 构建多模态请求（带关怀式提示词）
prompt = """请详细分析这张照片中的人物状态，告诉我：

1. **基本描述**：照片中的人物有什么外貌特征？
2. **表情和情绪**：人物的表情如何？传达出什么样的情绪或状态？
3. **穿着打扮**：人物穿着什么衣服？整体风格如何？
4. **环境背景**：照片的背景是什么？在什么场景下？
5. **光线和构图**：照片的光线如何？整体画面感觉如何？
6. **状态评估**：综合来看，这个人当前的精神状态如何？有什么建议吗？

⚠️ **重要回复要求**：
- 围绕用户的**状态、情绪、表情**来分析
- 用**温暖关心的语气**回复，像朋友一样
- 关注用户的感受，给予真诚的关心
- 不要冷冰冰的技术分析，要有人情味
- 如果用户看起来疲惫，给予休息建议
- 如果用户看起来状态好，给予鼓励
"""

payload = {
    "model": model,
    "messages": [
        {
            "role": "user",
            "content": [
                {
                    "type": "text",
                    "text": prompt
                },
                {
                    "type": "image_url",
                    "image_url": {
                        "url": f"data:image/png;base64,{image_base64}"
                    }
                }
            ]
        }
    ],
    "max_tokens": 4096,
    "temperature": 0.7
}

# 步骤 4: 发送请求
response = requests.post(
    f"{base_url}/chat/completions",
    headers=headers,
    json=payload,
    timeout=120
)

# 步骤 5: 解析结果
if response.status_code == 200:
    result = response.json()
    analysis = result['choices'][0]['message']['content']
    print(analysis)
```

## 📋 分析维度

qwen3-397b 模型可以从以下维度分析人像照片：

| 维度 | 分析内容 | 回复重点 |
|------|----------|----------|
| **1. 基本描述** | 性别、大致年龄、发型、面部特征 | 简洁描述，不过度聚焦外貌 |
| **2. 表情和情绪** ⭐ | 面部表情、眼神、情绪状态、动作姿态 | **重点分析**，回应用户情绪 |
| **3. 穿着打扮** | 服装类型、颜色、风格、配饰 | 简单带过，不评判 |
| **4. 环境背景** | 拍摄场景、背景物体、环境特征 | 帮助理解用户状态 |
| **5. 光线和构图** | 光源类型、光线方向、构图角度 | 简单提及 |
| **6. 状态评估** ⭐ | 精神状态、身体状态、工作/生活状态 | **重点分析**，给予关心 |
| **7. 关心建议** ⭐ | 休息建议、鼓励话语、温暖关怀 | **核心回复**，让用户感受到关心 |

## ⚠️ 注意事项

### 权限要求
- **摄像头权限**: macOS 需要用户授权终端/Python 访问摄像头
- **文件读写**: 需要写入 `~/Desktop/` 和读取照片的权限

### 依赖检查
```bash
# 检查 imagesnap 是否安装
which imagesnap

# 如果未安装，执行：
brew install imagesnap
```

### API 配置
确保以下配置可用：
- ✅ vLLM 服务运行中 (`http://10.72.0.44:8001/v1`)
- ✅ qwen3-397b 模型支持多模态输入（text + image）
- ✅ API Key 有效

### 隐私提示
⚠️ **重要**: 拍摄和分析人像照片涉及隐私，请确保：
1. 获得用户明确同意
2. 照片仅用于本次分析，不长期存储
3. 分析完成后及时清理临时文件

### 回复风格要求 ⭐
⚠️ **必须遵守**：
- ❌ 不要冷冰冰的技术分析
